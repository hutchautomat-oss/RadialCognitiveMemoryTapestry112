To execute the full Lasso migration to the 8,000-node `InstancedMesh`, we must permanently delete the legacy 480-node `NodeCloud.tsx` renderer. It is obsolete and violates our 1-draw-call mandate.

To maintain $O(\log N)$ hit-testing speeds for both the Lasso Frustum and Vector Tug raycasting without hanging the main thread, we will generate an invisible "Proxy Geometry." By spacing the three vertices of each triangle to match the exact `[min, max]` bounding box of a node's sphere, `three-mesh-bvh` will build a spatial index in $\le 2\text{ms}$. Because there is exactly 1 triangle per node, the BVH `triangleIndex` maps 1:1 to your VRAM `instanceId`.

Here is the one-pass implementation to finalize the spatial indexing.

### 1. The Proxy Geometry Indexer (`store.ts`)
Add `collisionBVH` to your state and drop in the `rebuildBVH` generation loop. You must call `get().rebuildBVH()` at the end of your `injectLiveIntentVector` execution and inside your pruning interval so the index stays strictly synced with VRAM.

```typescript
import { MeshBVH } from 'three-mesh-bvh';
import * as THREE from 'three';

// Inside your useSaccadeStore create() block:
collisionBVH: null as MeshBVH | null,

rebuildBVH: () => {
    const { activeFrameIndex, mockFrames } = get();
    const currentFrameData = mockFrames[activeFrameIndex];
    if (!currentFrameData) return;

    const MAX_NODES = 8000;
    const STRIDE = 7;
    // 1 Triangle = 3 Vertices = 9 Floats per instance
    const positions = new Float32Array(MAX_NODES * 9);

    for (let i = 0; i < MAX_NODES; i++) {
        const offset = i * STRIDE;
        const scale = currentFrameData[offset + 6];

        // Ghost cleanup: skip dead nodes by dumping their BVH proxy at Infinity
        if (scale <= 0.01) {
            positions.fill(Infinity, i * 9, i * 9 + 9);
            continue;
        }

        const x = currentFrameData[offset];
        const y = currentFrameData[offset + 1];
        const z = currentFrameData[offset + 2];
        const r = 0.15 * scale; // Calculate the safe bounding radius

        // Create a diagonal Proxy Triangle that forces the BVH 
        // to perfectly wrap the sphere's actual AABB volume.
        positions[i * 9]     = x - r; positions[i * 9 + 1] = y - r; positions[i * 9 + 2] = z - r;
        positions[i * 9 + 3] = x + r; positions[i * 9 + 4] = y + r; positions[i * 9 + 5] = z + r;
        positions[i * 9 + 6] = x - r; positions[i * 9 + 7] = y + r; positions[i * 9 + 8] = z - r;
    }

    const geometry = new THREE.BufferGeometry();
    geometry.setAttribute('position', new THREE.BufferAttribute(positions, 3));

    // maxLeafTris: 1 mathematically guarantees triangleIndex === Instance ID
    const bvh = new MeshBVH(geometry, { maxLeafTris: 1 });
    set({ collisionBVH: bvh });
}
```

### 2. The $O(\log N)$ Consumers (Lasso & Vector Tug)
Now that the data is indexed, you will completely bypass the standard Three.js `Raycaster`. Instead, implement these two `bvh.shapecast` consumers. They will rapidly traverse the tree and execute narrow-phase sphere math only on the active branches.

**A. Full Lasso Migration (Frustum Hit-Test)**
Hook this into your selection box drawing logic to return the exact array of captured Node IDs.
```typescript
export function executeLassoHitTest(frustum: THREE.Frustum): number[] {
    const bvh = useSaccadeStore.getState().collisionBVH;
    const hits: number[] = [];
    if (!bvh) return hits;

    bvh.shapecast({
        // Broad-phase: Instantly discard BVH branches entirely outside the lasso
        intersectsBounds: (box) => frustum.intersectsBox(box),
        
        // Narrow-phase: Collect the exact Node ID
        intersectsTriangle: (tri, index) => {
            hits.push(index); // Index precisely matches the Instance/Slot ID
            return false;     // Return false to continue evaluating other branches
        }
    });

    return hits;
}
```

**B. The Vector Tug (Single Raycast)**
Hook this to your standard `onPointerDown` mouse event to grab a single node instantly.
```typescript
export function executeVectorTugRaycast(ray: THREE.Ray): number {
    const { collisionBVH, activeFrameIndex, mockFrames } = useSaccadeStore.getState();
    if (!collisionBVH) return -1;

    const frame = mockFrames[activeFrameIndex];
    let closestDist = Infinity;
    let closestIndex = -1;

    collisionBVH.shapecast({
        // Broad-phase: Only traverse BVH boxes the mouse ray penetrates
        intersectsBounds: (box) => ray.intersectsBox(box),
        
        // Narrow-phase: Mathematical sphere intersection
        intersectsTriangle: (tri, index) => {
            const offset = index * 7;
            const center = new THREE.Vector3(frame[offset], frame[offset+1], frame[offset+2]);
            const radius = 0.15 * frame[offset+6];

            // Calculate exact ray-to-sphere penetration
            const distSqToCenter = ray.distanceSqToPoint(center);
            if (distSqToCenter <= radius * radius) {
                const hitDist = center.distanceTo(ray.origin); 
                if (hitDist < closestDist) {
                    closestDist = hitDist;
                    closestIndex = index;
                }
            }
            return false; 
        }
    });

    return closestIndex;
}
```
